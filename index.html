Tentang Saya

Saya adalah seorang Pelajar yang memiliki keahlian di bidang perangkat keras dan perangkat lunak komputer. Saya dapat merakit PC, menginstal Windows, memperbaiki CPU yang bermasalah, memperbaiki printer, dan mengerti HTML dan CSS.
Saya memiliki minat yang besar dalam bidang teknologi komputer. Saya selalu ingin mempelajari hal-hal baru tentang komputer dan bagaimana cara memperbaikinya. Saya juga senang membantu orang lain dengan masalah komputer mereka.
Saya memiliki pengalaman bekerja (PKL) sebagai teknisi komputer di sebuah service center selama 3 bulan, Selama bekerja di sana, saya telah menangani berbagai macam masalah komputer, mulai dari masalah perangkat keras hingga masalah perangkat lunak.

Karakter Saya

Saya adalah orang yang teliti, cermat, dan berorientasi pada hasil. Saya juga memiliki kemampuan komunikasi dan interpersonal yang baik.

Merakit PC

Saya memiliki keahlian dalam merakit komputer pribadi, mulai dari memilih komponen hingga memasangnya.

Menginstal Windows

Saya dapat menginstal sistem operasi Windows pada komputer pribadi dalam waktu kurang dari 1 Jam.

Memperbaiki CPU

Saya dapat mengatasi masalah pada CPU komputer pribadi, seperti overheating, blue screen of death, dan masalah perangkat keras lainnya.

Masalah Pada Printer

Saya dapat mengatasi masalah pada printer, seperti kertas macet, tinta habis, dan masalah perangkat lunak lainnya.

HTML & CSS

Saya memiliki pemahaman yang baik tentang bahasa markup HTML dan CSS, sehingga saya dapat membuat halaman web yang menarik dan responsif.

KOMPETEN DI LSP

tugas dan tanggung jawab


Crimping Kabel: Memasang konektor RJ45 dengan benar untuk transfer data yang efisien dan stabil dalam jaringan.

Membuat Topologi (Cisco): Memilih, mengkonfigurasi, dan menghubungkan perangkat jaringan sesuai desain topologi, serta memastikan koneksi dan fungsi jaringan sesuai spesifikasi.

Konfigurasi MikroTik: Menyesuaikan pengaturan jaringan, mengelola IP dan routing, konfigurasi firewall, keamanan, dan jaringan nirkabel, serta memastikan kinerja dan keamanan jaringan sesuai standar yang ditetapkan.
